Questão 1: o que é HTML

linguagem de programação padrão usada para criar a estrutura de uma página web, definindo o conteúdo e sua organização com elementos como títulos, parágrafos, imagens e links. Ao contrário de linguagens de programação, o HTML não é usado para criar funcionalidades, mas sim para estruturar o conteúdo de forma que navegadores possam interpretá-lo e exibi-lo corretamente.


Questão 2: Qual a diferença entr HTML e LINGUAGEM DE PROGRAMAÇÃO

A diferença principal é que HTML é uma linguagem de marcação que estrutura o conteúdo de páginas web, enquanto uma linguagem de programação executa lógica, processa dados e realiza ações. Linguagens de programação são imperativas e dão comandos para a máquina, usando variáveis e estruturas de controle, o que o HTML não faz.


Questão 3: Quais as funções das tags

As tags HTML definem a estrutura, o significado e a apresentação do conteúdo numa página web, dizendo ao navegador como interpretar e exibir textos, imagens, links, tabelas, e mais.

DOCTYPE HTML - É UMA DECLARAÇÃO QUE DIZ AO NAVEGADOR O TIPO DO DOCUMENTO

html lang=en inicia o html e diz qual idioma sera utilizado

head tag que é cabecalho e configurações do documento
